# Provenance

This directory is reserved for sanitized publication-reproduction manifests.
The final manifest is generated by:

```bash
python scripts/run_complete_climate_workflow.py
```

The sanitized manifests contain file names, sizes and SHA-256 checksums where
available, but no absolute local paths. The software environment used for the
release is recorded separately. Full local manifests are written to
`outputs/provenance/` and are not committed.
